玩家日常游戏中最常用的基础功能指令

/home mine

/warp ziyuan

/tpaccept /tpac 接受当前待处理的传送请求 /tpaccept
/tpdeny /tpno 拒绝当前待处理的传送请求 /tpdeny

/back 无 回到上一次死亡地点或传送前的最后一个位置 /back

RESIDENCE领地保护指令

保护你的建筑和财产，创建私人领地的基本指令

指令 描述 示例
/res select [x] [y] [z] 选择领地的长方体区域，X Y Z是从你当前位置为中心的距离。通常使用木锄选择更方便 /res select 10 5 10
/res select chunk 选取一整个区块（16×16）作为领地范围 /res select chunk
/res create [领地名] 选择好区域后创建一个叫做[领地名]的领地 /res create myhome
/res remove [领地名] 移除一个叫[领地名]的领地。不指定名称则移除当前所在的领地 /res remove myhome
/res subzone <领地名> [子区域名] 在领地内创建一块子区域，你必须是领地所有者 /res subzone myhome farm
/res list 显示你拥有的所有领地列表 /res list
/res info [领地名] 查看指定领地的详细信息。不指定名称则查看当前领地信息 /res info
/res info myhome
/res current 显示你当前所在的领地名称 /res current
/res pset <领地名> <玩家名> <权限> [true/false/remove] 设置某领地对于某玩家的权限（如build、use、container等） /res pset myhome Steve build true
/res set <领地名> <权限> [true/false/remove] 设置某领地的公共权限（对所有玩家生效） /res set myhome pvp false
/res tp [领地名] 传送到指定领地。不指定名称则传送到你拥有的第一个领地 /res tp myhome
/res tpset 设置当前领地的传送点为你站立的地方 /res tpset
/res give <领地名> <玩家名> 将某领地赠与某玩家，你必须是领主且被赠予玩家在线 /res give myhome Steve
/res market buy [领地名] 购买在售的某领地。不指定名称则购买当前查看的领地 /res market buy shop1
/res market list 显示所有在售的领地列表 /res market list

重要提醒

1. 使用木锄选择领地范围是最方便的方法：左键点击第一个点，右键点击对角点

2. 新玩家默认可创建的领地数量有限，VIP玩家可获得更多领地权限

3. 领地创建后默认只有所有者有全部权限，需要手动给朋友添加权限

4. 领地权限包括：build(建造)、use(使用工作台等)、container(使用箱子)、tp(传送)等

QUICKSHOP商店交易指令

创建和管理个人商店，与其他玩家进行物品交易

指令 描述 示例
创建商店方法 1. 放置一个箱子
2. 手持要售卖/收购的物品，左击箱子
3. 输入价格
4. 放入商品（出售商店）或放钱（收购商店）		 手持钻石左击箱子
输入 100（每个钻石100游戏币）
放入钻石到箱子
/qs buy 将鼠标准星所对的商店转变为收购商店（收购玩家物品） 瞄准商店牌子输入 /qs buy
/qs sell 将鼠标准星所对的商店转变为出售商店（出售物品给玩家） 瞄准商店牌子输入 /qs sell
/qs price <价格> 将鼠标准星所对的商店的交易价格修改为指定值 瞄准商店牌子输入 /qs price 50
/qs find <物品> 找到(最近的)交易指定物品的商店，支持模糊搜索 /qs find diamond
/qs find 钻石
/shop /qs 的指令别名，功能相同 /shop find iron
如何使用商店 1. 右键点击商店箱子
2. 在聊天栏输入购买数量
3. 自动完成交易（出售商店：用钱买物；收购商店：用物换钱）		 右键点击商店
输入 5（购买5个）
自动扣除250游戏币，获得5个商品

商店使用提示

1. 商店必须放置在领地内或公共市场区域，否则可能被清除

2. 出售商店：箱子内放商品，玩家用游戏币购买商品

3. 收购商店：箱子内放游戏币，玩家用商品兑换游戏币

4. 使用 /qs find 命令可以快速找到需要的物品，支持中文和英文搜索

5. 商店价格单位是"每个物品的价格"，不是总价

COREPROTECT区块查询指令

查询方块和容器的操作记录，用于追踪破坏和盗窃行为

指令 别名 描述 示例
/co inspect /co i 切换检查模式。开启后，点击方块或实体可快速查看其变更历史 /co i
然后点击被破坏的方块
/co lookup /co l 查询数据。根据参数查询方块、交互等历史记录。默认查询范围为全服 /co lookup u:Steve t:1d
/co near 无 查询玩家周围5格内的历史记录，快速查看附近发生的事 /co near
/co help 无 显示CoreProtect命令列表和帮助信息 /co help
查询参数示例 无 u:玩家名 - 按用户筛选
t:时间 - 按时间筛选
r:范围 - 按范围筛选
a:动作 - 按动作类型筛选		 /co lookup u:Steve t:2h r:10 a:block

CoreProtect查询参数详解

u:<玩家名>
按用户筛选，可指定单个或多个用户，如 u:Steve 或 u:Steve,Alex
t:<时间>
按时间范围筛选，支持周(w)、天(d)、时(h)、分(m)、秒(s)，如 t:2h 或 t:1d
r:<范围>
按空间范围筛选，数字表示半径，如 r:10 表示周围10格
a:<动作>
按动作类型筛选，如 a:block(方块放置/破坏) 或 a:container(容器存取)
i:<包含>
仅包含特定方块/实体，如 i:diamond_ore 只查询钻石矿相关记录

AUTHME账户安全指令

账户注册、登录和安全设置相关指令，保护你的账户安全

指令 描述 示例
/register [密码] [确认密码] 注册账户。首次进入服务器必须注册账户才能进行游戏 /register mypassword123 mypassword123
/login <密码> 登录账户。每次进入服务器都需要登录验证身份 /login mypassword123
/logout 登出账户。安全退出游戏，防止他人使用你的账户 /logout
/changepassword <旧密码> <新密码> 修改账户密码，建议定期更换密码保证安全 /changepassword oldpass newpass123
/unregister <密码> 解除账户注册，删除账户数据。此操作不可逆！ /unregister mypassword123
/email show 显示你当前绑定的邮箱地址 /email show
/email add <邮箱> <确认邮箱> 给你的账户添加一个新邮箱，用于找回密码 /email add example@email.com example@email.com
/email recover <邮箱> 通过邮箱找回密码，系统会向邮箱发送新密码 /email recover example@email.com

账户安全提示

1. 首次进入服务器必须注册账户，否则无法移动和进行任何操作

2. 每次进入服务器都需要登录，确保账户安全

3. 密码不要使用过于简单的组合，建议包含字母、数字和特殊字符

4. 强烈建议绑定邮箱，以便忘记密码时能够找回账户

5. 不要在公共场所或不安全的网络环境下登录游戏账户

6. 离开电脑时务必使用 /logout 命令登出账户
