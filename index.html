玩家指令手册

本页面整理了Dream World服务器中玩家可使用的所有指令。按照插件分类，便于快速查找和学习。点击下方分类按钮可快速跳转到对应部分。

Residence - 领地管理

保护你的建筑和领地，管理权限与设置

重要提示
领地是保护你建筑和财产的重要系统。使用木锄（默认工具）选择区域后，使用/res create创建领地。 所有命令不写领地名时，默认为当前所在的领地。
指令 参数 描述
/res select [x] [y] [z] 选择领地区域，X Y Z是从当前位置为中心的距离
/res select chunk 选取一整个chunk区域
/res select expand [格数] 向你的前方延伸选区
/res select vert 把选区延伸到从天顶到地底
/res create [领地名] 选择好区域后创建一个叫做[领地名]的领地
/res remove [领地名] 移除一个叫[领地名]的领地
/res subzone <领地名> [子区域名] 在领地内创建一块子区域，你必须是所有者才行
/res info [领地名] 得到某领地的信息
/res list 显示你拥有的领地
/res current 显示你所在的领地
/res set [领地名] [权限] [true/false/remove] 设置某领地的权限
/res pset [领地名] [玩家名] [权限] [true/false/remove] 设置某领地对于某玩家的权限
/res gset [领地名] [群组名] [权限] [true/false/remove] 设置某领地对于某群组的权限
/res give [领地名] [玩家名] 将某领地赠与某玩家，你必须是领主且被赠予玩家在线
/res message [领地名] [enter/leave] [信息] 设置进入/退出某领地时候显示的信息
/res tp [领地名] 传送至某领地
/res tpset 设置当前领地的传送点为你站立的地方
/res market list 显示在售的所有领地
/res market buy [领地名] 购买某领地
/res market sell [领地名] [价格] 将某领地出售
常用权限说明
build - 建造权限(包括destroy和place) | container - 是否能使用箱子，发射器等 | use - 使用权限(工作台，炉子等) | tp - 传送权限 | move - 进入权限 | pvp - 设置是否能PVP

EssentialsX - 基础功能

玩家日常使用的传送、家园、经济等功能

指令 别名 描述
/home [家名称] 传送到你的家
/sethome [家名称] 设置你的家
/delhome [家名称] 删除你的家
/warp [传送点名称] 传送到公共传送点
/tpa <玩家名> 请求传送到某玩家
/tpahere <玩家名> 请求某玩家传送到你
/tpaccept /tpac 接受传送请求
/tpdeny /tpno 拒绝传送请求
/spawn 传送到主城出生点（还没有）
/back 回到上一次死亡地点或传送地点
/bal [玩家名] /balance, /money 查看余额
/pay <玩家名> <金额> 向玩家支付金额
/msg <玩家名> <信息> /m, /tell, /whisper 私聊某玩家
/r <信息> /reply 回复最后私聊你的人
/mail read 查看邮件
/mail send <玩家名> <信息> 发送邮件
/afk 标记自己为挂机状态
/nick <昵称> 设置你的昵称
/hat 将手中的物品戴在头上

QuickShop - 商店经济

创建和管理商店，进行物品交易

如何创建商店
1. 将一个箱子放在地上
2. 手持你要售卖或收购的物品，左击箱子
3. 在聊天栏中输入物品的价格
4. 将物品放入箱子中
指令 描述
/qs unlimited 将鼠标准星所对的商店库存设置为"无限"
/qs setowner <player> 将商店的所有人改为指定玩家
/qs buy 将鼠标准星所对的商店转变为收购商店
/qs sell 将鼠标准星所对的商店转变为出售商店
/qs price <price> 将鼠标准星所对的商店的交易价格修改为指定值
/qs clean 将所有库存数量为0的商店移除
/qs find <item> 找到(最近的)交易指定物品的商店，例如 '/qs find dia' 将会找到最近的交易钻石的商店
/shop /qs的指令别名

AuthMe - 账户安全

账户注册、登录和安全相关指令

指令 描述
/login <password> 登录账户
/register [password] [verifyPassword] 注册账户
/logout 登出账户
/changepassword <oldPassword> <newPassword> 更改密码
/unregister <password> 解除账户注册
/email show 显示你当前的邮箱
/email add <email> <verifyEmail> 给你的账户添加一个新邮箱

/captcha <captcha> 输入验证码完成验证

CoreProtect - 日志查询

查询方块操作和玩家行为记录

使用说明
CoreProtect记录服务器的所有方块操作和玩家行为。玩家可以查询自己的操作记录，管理员可以查询全服记录。 常用参数：u:用户 t:时间 r:范围 a:动作类型 i:包含的方块 e:排除的方块
指令 别名 描述
/co help 显示命令列表
/co inspect /co i 切换检查模式。开启后，点击方块或实体可快速查看其变更历史
/co lookup /co l 查询数据。根据参数查询方块、交互等历史记录
/co near 执行附近查询。相当于 /co lookup r:5，查询玩家周围5格内的历史记录
常用查询示例
/co lookup u:你的名字 t:1d - 查询你过去1天内的所有操作
/co lookup u:你的名字 a:block t:2h - 查询你过去2小时内所有方块操作
/co lookup i:diamond_ore a:-block t:6h - 查询过去6小时内被挖走的钻石矿记录
/co near - 查询你周围5格内的历史记录

使用提示

1. 大部分指令可以使用Tab键自动补全，节省输入时间

2. 指令中的<参数>表示必填参数，[参数]表示可选参数

3. 如果遇到指令无法使用的情况，可能是权限不足或指令格式错误

4. 建议将常用指令添加至快捷栏或记忆快捷键

5. 更多问题可联系管理员或在QQ群中询问
